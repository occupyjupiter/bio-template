# Pink Bio Card

A simple animated bio card with falling cherry blossom petals. Designed to showcase your profile with social media links and GitHub, TikTok, Telegram, etc.

## Features

* ** Floating cherry blossom petals ** animation in the background
* ** Profile card ** with avatar, username, and slogan
* **Social media links:** Telegram, GitHub, TikTok
* **Adaptive design** - works on mobile devices and desktop computers
* Clean, pastel pink theme
